# Responsive Rules

Always design mobile-first.

Support:

- Mobile
- Tablet
- Desktop

Patterns:

Mobile:
- Single column
- Stacked layout

Tablet:
- Two-column adaptation

Desktop:
- Multi-column grid

Avoid:

- Fixed widths
- Desktop-first layouts
- Pixel-perfect assumptions

Validate:

- Readability
- Accessibility
- CTA visibility
- Navigation discoverability
