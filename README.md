# Agent Skills

A collection of agent skills for indie builders.

## Branding & Naming

- **indie-app-naming** - Best practices for choosing a strong, memorable, and available name for an indie app or game.
  ```bash
  npx skills@latest add donleqt/skills/indie-app-naming
  ```
