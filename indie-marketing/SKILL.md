# Indie Marketing Skill

## When to Use This Skill
- User wants realistic marketing and growth advice for an indie app, tool, micro-SaaS, or game.
- User asks for a growth plan, launch strategy, or ways to acquire first users organically.
- Especially useful when the user wants **actionable copy and ready-to-use materials**.

## Core Mindset (2026)
- Distribution > Features.
- Give value first -> promotion second.
- Quality + consistency beats volume.
- Build in public and community participation are the strongest growth levers for solo founders.
- Focus on 2-3 channels maximum.

## Step-by-Step Process the Skill Must Follow
1. Understand the project context (from files, CLAUDE.md, or user input).
2. Define clear positioning and value proposition.
3. Recommend 2-3 realistic core channels.
4. Create a short-term actionable plan (usually 30 days).
5. Generate **practical, copy-paste-ready materials**.
6. End with clear next steps.
7. Always instruct the user to save the output.
